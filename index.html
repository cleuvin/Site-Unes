Sobre o desenvolvimento deste site.

Este é um site exemplo, feito através de um treinamento onde foi desenvolvido apenas utilzando o HTML5 puro, sem estilizações. Foi adicionado imagens e bordas, vendo o codigo fonte você terá a noção de como foi desenvolvido usando apenas o HTML5 para fazer o alinhamento das imagens, os links e os textos, sem nenhuma utilização de CSS ou demais.
